# Domain Knowledge — Pharmacovigilance

> the domain concepts to understand what the code does.

## Reading Order

1. **[pharmacovigilance_101.md](pharmacovigilance_101.md)** — Start here.
   All PV concepts explained in plain English: adverse events, seriousness,
   MedDRA, causality, FAERS, and every term you'll see in the code.

2. **[faers_schema.md](faers_schema.md)** — Read this when you start
   the data pipeline. Every FAERS table, every column, every type, and

3. **[worked_examples.md](worked_examples.md)** — Read this to see
   exactly what the model's input and output looks like. Contains a
