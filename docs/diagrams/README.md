# Architecture Diagrams

Pipeline architecture diagrams in Mermaid (`.mmd`) and rendered SVG (`.svg`).

## Available Diagrams

### Data Pipeline
| File | Description |
|------|-------------|
| `data_pipeline_architecture.mmd` / `.svg` | High-level data pipeline architecture |
| `data_pipeline_ppt.mmd` / `.svg` | Presentation-friendly data pipeline overview |
| `data_pipeline_simple.mmd` / `.svg` | Simplified data pipeline summary |

### Training Pipeline
| File | Description |
|------|-------------|
| `training_pipeline.mmd` / `.svg` | Training pipeline overview (SFT + WiSE-FT) |

### Evaluation Pipeline
| File | Description |
|------|-------------|
| `evaluation_pipeline.mmd` / `.svg` | Gemmra-Bench evaluation pipeline |

## Usage

- `.mmd` files can be edited and rendered with any Mermaid-compatible tool
- `.svg` files are pre-rendered for direct embedding in documentation or presentations
- Use plain Mermaid-safe labels to avoid parser issues in stricter renderers
